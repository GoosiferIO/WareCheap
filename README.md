# warecheap

WareCheap is a Flutter project that helps you find the cheapest grocery prices in your local community.

## Technologies

- WareCheap leverages Dart and Flutter as the base 

### Linter
To use the Linter do the commands:
- "Flutter pub get"
- "dart analyze"
